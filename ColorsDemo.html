How to set colors to UR HTML Pages?

We use styles for the HTML tags to set colors to the pages and its contents. style is a html attribute that can be set in different ways.

Styles

Styles provide a way of presenting the content of the HTML is a neat manner. This might include colors, locations of the content along with sizes defined.
There are 3 different ways of providing styles to UR HTML Tags. They are:

  1. Inline Styling
  2. Styling as a HTML Tag
  3. External Styling thro CSS

Inline Styles

Inline styles can be set to the specific HTML tag by using an attribute called style. U can fill the multiple styles by providing the name: value seperated by ;
The style that is set is applicable only to the tag to which the attribute style is set

In File Styles

Alternatively, U could set styles to the whole HTML Document using a style tag. The style tag is placed in the head section of the HTML document.
